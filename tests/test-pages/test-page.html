Test Xəbər Məqaləsi - LNK Genişlənməsi Testi

Nəşr: Test News | Tarix: 2025-01-27 | Müəllif: Test Müəllifi

Bu, LNK Media Bias Analyzer genişlənməsinin işləyib-işləmədiyini yoxlamaq üçün yaradılmış test məqaləsidir. Genişlənmə bu səhifəni xəbər məqaləsi kimi tanımalı və təhlil etməyə imkan verməlidir.

Genişlənmə aşağıdakı funksionallığı təmin etməlidir:

  • Səhifədə yaşıl göstərici göstərmək
  • Genişlənmə ikonuna kliklədikdə popup açmaq
  • "Təhlil et" düyməsi ilə məqaləni təhlil etmək
  • Etibarlılıq və siyasi meyl balını göstərmək
  • Tam təhlil üçün LNK.az-a yönləndirmək

Əgər genişlənmə düzgün işləyirsə, siz səhifənin sağ üst küncündə yaşıl göstərici görəcəksiniz. Genişlənmə ikonuna kliklədikdə popup açılacaq və məqaləni təhlil edə biləcəksiniz.

Bu test məqaləsi müxtəlif xəbər saytlarının strukturunu simulyasiya edir və genişlənmənin müxtəlif saytlarda işləyib-işləmədiyini yoxlamaq üçün istifadə edilə bilər.
